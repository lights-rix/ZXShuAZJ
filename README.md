# 项目简介

## 核心功能
- 基于 YOLOv8 的生猪检测系统
- 支持图片、视频及摄像头进行检测
- 提供完整的 UI 界面，可实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并直接查看所有检测结果
- 支持检测结果的保存功能
- 支持自定义置信度和 IOU 阈值
- 可选择显示标签和原图
- 可选择不同的检测模型

## 适用场景
- 生猪养殖场自动化检测
- 农业科研中的生猪行为分析
- 畜牧业管理中的数量统计
- 农业智能化管理系统集成

## 技术特点
- 采用 Python 3.9 + OpenCV + PyQt5 + Ultralytics + Torch 1.9 技术栈
- 包含 1000 张标注数据集，专注于 ['pig'] 单类检测
- 提供完整的源代码、训练好的模型、数据集和运行教程
- 支持多种输入方式（图片、视频、摄像头）
- 已训练完成的模型，配置成功环境可直接使用
